## Intro

本项目基于 PythonWeb 的 C/S 微架构（Flask），实现了教务管理系统（Management System）

- 涉及到的知识点有：面向对象程序设计、文件调用关系等
- 涉及到的技术栈有：Flask、Html、Css、JavaScrip、MySQL
